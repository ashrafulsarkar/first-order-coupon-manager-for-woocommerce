Maintain the first-order discount using this plugin.

== Description ==
“First Order Coupon Manager for WooCommerce” enables admin to offer discounts to their customers on their first order with various conditions like the minimum amount they need to spend to get a discount or whether users can use multiple coupons together.

== Installation ==
1. Deactivate the plugin if you have the previous version installed.
2. Extract “first-order-coupon-manager-for-woocommerce.zip” archive content to the “/wp-content/plugins/first-order-coupon-manager-for-woocommerce” directory.
3. Activate “First Order Coupon Manager for WooCommerce” plugin via “Plugins” menu in WordPress admin menu.
4. Create coupons and use them according to your needs.

== Frequently Asked Questions ==
= How do I setup this plugin? =
* To setup the plugin, go to wp-admin > WooCommerce > Coupon
= Do we collect user information through this plugin?
* No, we do not collect any data through this plugin.
